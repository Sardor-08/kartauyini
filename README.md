# Card Memory Game

## Overview
This Card Memory Game is a fun and engaging application developed in Kotlin for the Android platform. The game is designed to improve memory skills by challenging players to match pairs of cards.

## Features
- **User-friendly interface**: The game offers an intuitive user interface that is easy to navigate.
- **Different difficulty levels**: Players can choose from various difficulty levels to match cards.
- **Timer and score tracking**: The game keeps track of the time taken and the score to enhance competitiveness.
- **Sound effects and animations**: Enjoyable sound effects and animations to enhance gameplay experience.

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/Sardor-08/kartauyini.git
   ```
2. Open the project in Android Studio.
3. Run the application on an Android device or emulator.

## Technologies Used
- **Kotlin**: The primary programming language for development.
- **Android SDK**: For building Android applications.
